




📚 Book Search Engine
 ## Description :

A Google Books search engine built with the MERN stack, refactored from RESTful API to GraphQL using Apollo Server. Search for books, save your favorites, and manage your personal reading list.

## Usage :

    Search - Enter a book title to see results from Google Books

    Login/Signup - Create an account or log in to save books

    Save Books - Click "Save this Book!" to add to your collection

    View Saved - Navigate to "See Your Books" to manage your saved titles

    Delete - Remove books from your saved list anytime

## Technologies :

    MongoDB + Mongoose

    Express.js

    React.js

    Node.js

    GraphQL + Apollo Server

    JSON Web Tokens

    Bootstrap
